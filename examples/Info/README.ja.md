# Info サンプル

[English README](README.md)

Infoカテゴリでは、Arduino-ESP32のビルド/実行時設定を確認できるヘルパーのサンプルを掲載します。

- **BoardSettings (`EspHelperBoardSettings.h`)** – USB設定、CPU周波数、フラッシュ/PSRAM、パーティション構成などを起動時にSerialへ出力します。 (Sketch: [BoardSettings/BoardSettings.ino](BoardSettings/BoardSettings.ino))

詳細は `BoardSettings/` を参照してください。
